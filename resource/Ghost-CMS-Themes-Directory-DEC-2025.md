# Ghost CMS Themes & Tools Directory (2022–2025)

## Introduction

This document is a curated directory of Ghost CMS theme repositories and essential tools/resources for theme development, validation, localization, and deployment. Organized to facilitate browsing by origin and category, identify popular themes, and discover tooling for building and maintaining Ghost themes.

**Note:** GitHub star counts are snapshots from late December 2025 (where available).

## NOTES FOR THEME SELECTION

1. **Official TryGhost themes** are regularly maintained and well-documented.
2. **Community themes** vary in maintenance; check last commit date and issue activity.
3. **Popular themes** (1k+ stars) have larger user bases and active communities for support.
4. **Minimal themes** prioritize typography and performance; suitable for blogs and newsletters.
5. **Magazine themes** offer rich media layouts; ideal for publications and image-heavy content.
6. **Specialized themes** (courses, podcasts, portfolios) may include built-in features for those use cases.
7. Most themes use **Handlebars** templating; some newer ones support **Tailwind CSS** and build tools like **Vite**.
