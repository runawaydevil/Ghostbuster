## About

**Ghostbuster** is a curated directory of Ghost CMS theme repositories and essential tools/resources for theme development, validation, localization, and deployment.

This website provides a comprehensive, organized reference for browsing Ghost themes by origin and category, identifying popular themes, and discovering tooling for building and maintaining Ghost themes.

## What's Included

The directory contains:

- **Official TryGhost Themes** - Regularly maintained and well-documented themes from the Ghost team
- **Popular Community Themes** - Highly-rated themes from the community
- **Responsive & Minimalist Themes** - Clean, mobile-optimized designs
- **Developer & Programming Themes** - Themes tailored for technical content
- **Specialized/Niche Themes** - Themes for specific use cases (podcasts, portfolios, courses, etc.)
- **Theme Collections** - Bundled theme packs
- **Tools & Resources** - Essential development tools, documentation, and utilities
